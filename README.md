# LocateIQ
LocateIQ is an AI-powered location intelligence system that helps investors identify the most suitable investment locations in the Asir region using spatial analysis, GIS technologies, and machine learning.
#My Contribution

My primary responsibility in the project was Data Engineering, including collecting and preparing the data, designing the database, performing spatial analysis, and building the final dataset used by the AI recommendation model.

This repository contains only my individual contribution to the project.
